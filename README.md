Para executar o projeto, basta digitar `make run`. Caso queira garantir sempre a integridade do build, basta digitar `make clean && make run`. Recomenda-se que tenha instalado no mínimo o C++ 11 e o make.

Após a execução do programa, será gerado um arquivo results.txt com todo o passo a passo que foi executado durante a normalização da gramática.
